# hpsdr-rpitx
Trasceiver based on:
- librpitx (https://github.com/F5OEO/librpitx)
- hpsdrsim (https://github.com/hiperiondev/hpsdrsim)

*COMPILE PROJECT:*
- git clone https://github.com/hiperiondev/hpsdr-rpitx.git
- cd hpsdr-rpitx/build
- make

*USE:*
hpsdr-rpitx [flags]

flags:
 - -atlas:  
 - -hermes: 
- -griffin: 
- -angelia: 
- -orion: 
- -orion2: 
- -hermeslite:
- -hermeslite2: 
- -c25 :
- -diversity:  
- -P1:
- -P2:
- -debug:
